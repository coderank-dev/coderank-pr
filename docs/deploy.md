# Deployment guide

This document is a placeholder. Full deployment guide lands alongside the first working deployment.

## Target environment

- **Google Cloud project** with the following APIs enabled: Vertex AI, Agent Engine, Cloud Run, Cloud Build, Pub/Sub, Secret Manager, Artifact Registry, Cloud Trace, Cloud Logging.
- **GitHub App** installed on the repositories you want reviewed.
- **Agents CLI** installed: `uvx google-agents-cli setup`.

## One-command deploy (target state)

```bash
agents-cli infra single-project
agents-cli deploy
```

Infrastructure is managed via Terraform generated by `agents-cli scaffold enhance`. CI/CD runs via GitHub Actions and pushes to Agent Runtime on merge to `main`.
